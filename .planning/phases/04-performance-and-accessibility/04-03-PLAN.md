  <name>Task 1: Performance E2E tests — fonts, images, gradients</name>
  <files>e2e/performance.spec.ts</files>
  <action>
Create `e2e/performance.spec.ts` with the following test cases:

1. **No Google Fonts network requests (UIUX-06):**
   - Navigate to homepage while intercepting network requests via `page.on('request', ...)`
   - Collect all request URLs
   - Assert zero requests to `fonts.googleapis.com` or `fonts.gstatic.com`
   - This proves fonts are self-hosted

2. **Self-hosted font files are served (UIUX-06):**
   - Make requests to `/fonts/plus-jakarta-sans-400-latin.woff2`, `/fonts/plus-jakarta-sans-700-latin.woff2`, `/fonts/jetbrains-mono-400-latin.woff2`
   - Assert all respond with 200 status
   - Assert content-type includes `font/woff2` or `application/font-woff2`

3. **Font preload link exists in head (UIUX-04):**
   - Navigate to homepage
   - Assert `link[rel="preload"][href*="plus-jakarta-sans-400"]` exists in the document head
   - Assert it has `as="font"`, `type="font/woff2"`, and `crossorigin` attributes

4. **OG image is not a raw PNG in root (UIUX-05):**
   - Request `/og.png` and assert it returns 404 (moved to src/assets/ for Astro optimization)
   - Assert that the `meta[property="og:image"]` tag exists with a non-empty `content` attribute

5. **Images have width and height attributes (UIUX-05):**
   - Navigate to homepage
   - Find all `<img>` elements
   - Assert each has `width` and `height` attributes (prevents CLS)

Import from `@playwright/test` (not axe-test fixture — these are performance tests, not accessibility scans).

e. **Skip link works (UIUX-08):** - Navigate to homepage - Press Tab (first focusable element should be skip link) - Press Enter - Assert focus or scroll target is `#main`
