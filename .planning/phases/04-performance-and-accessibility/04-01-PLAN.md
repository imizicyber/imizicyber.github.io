   - `plus-jakarta-sans-400-latin.woff2` (Plus Jakarta Sans regular 400)
   - `plus-jakarta-sans-700-latin.woff2` (Plus Jakarta Sans bold 700)

2. Add 3 `@font-face` declarations at the TOP of `src/styles/global.css` (before any other rules, before :root). Use the exact pattern from 04-RESEARCH.md:
   - font-family: 'Plus Jakarta Sans' for weights 400 and 700
   - font-family: 'JetBrains Mono' for weight 400
   - All use `font-display: swap`
   - All use `format('woff2')`
   - Include `unicode-range` for Latin characters per RESEARCH.md

3. In `src/layouts/BaseLayout.astro`:
   - REMOVE the 3 Google Fonts lines (preconnect to fonts.googleapis.com, preconnect to fonts.gstatic.com, stylesheet link) — lines 89-94
   - ADD a single preload link for the critical body font ONLY (not all 3):
     ```html
     <link
       rel="preload"
       href="/fonts/plus-jakarta-sans-400-latin.woff2"
       as="font"
       type="font/woff2"
       crossorigin
     />
     ```
     Place it where the Google Fonts links were. The `crossorigin` attribute is REQUIRED even for same-origin fonts (without it, browser downloads the font twice).

4. Verify: `npm run build` succeeds. Check that `dist/fonts/` contains the 3 WOFF2 files. Check that NO reference to fonts.googleapis.com or fonts.gstatic.com exists in the built output.
